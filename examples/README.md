# Pulse Examples

This directory contains examples demonstrating various features of Pulse.

## Basic Examples

- `basic.ts` - Core reactivity concepts (signals, computed, effects, stores)

## Advanced Examples

- `advanced.ts` - Advanced features including:
  - History & Undo/Redo
  - Form Validation
  - WebSocket Integration
  - Redux-like Reducers
  - State Machines
  - Animations
  - Middleware
  - Persistence
  - Async Computed
  - Batch Updates

## Framework Integration Examples

- `react-example.tsx` - React hooks integration
- `vue-example.vue` - Vue 3 composables integration

## Running Examples

### TypeScript Examples

```bash
# Using tsx
npx tsx examples/basic.ts
npx tsx examples/advanced.ts

# Using ts-node
npx ts-node examples/basic.ts
```

### React Example

```bash
# In a React project
npm install pulse
# Then import and use the components
```

### Vue Example

```bash
# In a Vue 3 project
npm install pulse
# Then use the composables in your components
```

## Example Topics

1. **State Management** - Managing application state with signals
2. **Form Handling** - Reactive forms with validation
3. **Real-time Updates** - WebSocket and async data
4. **Animation** - Signal-driven animations
5. **Persistence** - Saving state to localStorage
6. **Performance** - Batching and optimization techniques
7. **Framework Integration** - Using Pulse with React/Vue
